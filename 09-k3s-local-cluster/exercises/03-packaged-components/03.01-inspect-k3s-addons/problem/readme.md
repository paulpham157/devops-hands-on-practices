# Inspect K3s AddOns

## Task

Run:

```bash
cd 09-k3s-local-cluster
./scripts/inspect-addons.sh
```

Then answer:

1. Which namespace contains the packaged system pods?
2. What is the IngressClass name?
3. What is the default StorageClass?
4. Is metrics-server running?

## Done When

You can identify the components that make K3s feel ready out of the box.

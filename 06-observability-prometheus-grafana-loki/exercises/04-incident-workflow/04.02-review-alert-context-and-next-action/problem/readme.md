# Review Alert Context and Next Action

## Task

Assume an alert fired for elevated error rate on `course-app`.

Write a short response plan that includes:

1. Which PromQL or dashboard you would open first.
2. Which Loki query would confirm matching errors.
3. What evidence would separate app failure from downstream dependency failure.
4. One clear next action if the alert is real.

## Done When

You can turn an alert into an investigation starting point instead of treating the alert text as the whole diagnosis.

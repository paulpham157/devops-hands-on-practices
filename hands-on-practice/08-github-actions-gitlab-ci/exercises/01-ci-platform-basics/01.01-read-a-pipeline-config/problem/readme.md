# Read a Pipeline Config

## Task

Compare the two pipeline files:

```text
github-actions/workflows/python-docker-ci.yml
gitlab-ci/.gitlab-ci.yml
```

Answer these questions in your own notes:

1. What events trigger the GitHub Actions workflow?
2. Which GitHub Actions job waits for another job?
3. What are the GitLab CI stages?
4. Which GitLab CI job creates a JUnit artifact?
5. Which platform file explicitly defines Docker-in-Docker?

## Done When

You can explain the order of jobs in both files without running them.

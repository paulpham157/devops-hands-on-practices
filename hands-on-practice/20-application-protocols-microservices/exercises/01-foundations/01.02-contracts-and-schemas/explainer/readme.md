# Explainer: Contracts and Schemas

Contracts describe how services communicate.

This lesson includes:

| Contract | File |
| --- | --- |
| OpenAPI | `rest-api/openapi.yaml` |
| Protobuf | `grpc-service/proto/catalog.proto` |
| GraphQL schema | `protocol-specs/schema.graphql` |
| AsyncAPI | `protocol-specs/asyncapi.yaml` |

Contracts help teams review changes, generate clients, document APIs, and detect breaking changes.

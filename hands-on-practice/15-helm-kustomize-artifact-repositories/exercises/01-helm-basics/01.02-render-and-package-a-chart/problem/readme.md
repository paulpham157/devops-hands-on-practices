# Problem: Render and Package a Chart

From the lesson directory, run:

```bash
./scripts/render-helm.sh
./scripts/package-chart.sh
```

Then answer:

1. Which file contains the rendered Kubernetes YAML?
2. Which file contains the packaged chart?
3. What version is used in the package name?
4. Why should CI publish a chart package instead of relying on a developer laptop?
5. What command would install version `0.1.0` from an internal Helm repository?

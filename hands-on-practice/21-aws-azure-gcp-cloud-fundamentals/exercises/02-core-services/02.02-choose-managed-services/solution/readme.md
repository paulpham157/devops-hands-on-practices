# Solution: Choose Managed Services

1. Managed PostgreSQL. Backups and patching matter, and the team is small.
2. Self-managed may be needed if the managed service cannot support the extension.
3. Managed object storage. S3, Blob Storage, and Cloud Storage are strong defaults.
4. Managed Kubernetes. EKS, AKS, or GKE avoids self-managing the control plane.
5. Self-managed VM or appliance model. Special kernel modules usually do not fit serverless or managed app platforms.

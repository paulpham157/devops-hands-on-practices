# Choose an IaC Tool for a Team

Choose one primary tool for each team.

1. AWS-only backend team, already strong in TypeScript, wants reusable constructs.
2. Multi-cloud platform team managing AWS, Cloudflare, GitHub, and Datadog.
3. Small AWS team wants the fewest moving parts and native stack rollback.
4. Enterprise team already has Terraform remote state, modules, and policy checks.

For each answer, explain the tradeoff.

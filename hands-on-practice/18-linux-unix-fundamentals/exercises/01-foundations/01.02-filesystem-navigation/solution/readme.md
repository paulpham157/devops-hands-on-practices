# Solution: Filesystem Navigation

In this lab, the current working directory should usually be:

```text
/workspace
```

OS configuration usually lives under:

```text
/etc
```

Linux process and kernel information is exposed through:

```text
/proc
```

Temporary practice files should go under:

```text
/tmp
```

Alpine is smaller than Ubuntu. Some commands, shells, packages, or flags may be missing or implemented by BusyBox.

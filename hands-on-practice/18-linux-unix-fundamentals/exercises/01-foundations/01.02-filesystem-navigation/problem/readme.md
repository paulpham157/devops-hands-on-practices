# Problem: Filesystem Navigation

Start the lab:

```bash
./scripts/start-lab.sh
./scripts/shell-ubuntu.sh
```

Inside Ubuntu, run:

```bash
pwd
ls -la /
ls -la /etc | head
ls -la /proc | head
df -h
```

Answer:

1. What is your current working directory?
2. Which directory contains OS configuration?
3. Which directory exposes process and kernel information on Linux?
4. Which directory should you use for temporary practice files?
5. What is different when you run similar commands in Alpine?

# Explainer: Linux vs Unix vs POSIX

Unix is a historical operating system family and a design tradition.

Linux is a Unix-like kernel. A Linux distribution combines the Linux kernel with userland tools, libraries, package managers, and system services.

GNU/Linux usually means a Linux system with GNU userland tools such as Bash and GNU Coreutils.

POSIX is a standards family for portable operating system interfaces, including shell and utility behavior.

For DevOps, the practical point is simple: scripts and commands may behave differently between GNU/Linux, BusyBox, BSD, macOS, and commercial Unix systems.

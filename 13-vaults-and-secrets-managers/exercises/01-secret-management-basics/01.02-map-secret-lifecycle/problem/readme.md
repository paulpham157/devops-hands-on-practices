# Problem: Map a Secret Lifecycle

Use this example:

```text
A web app needs an API token for a payment provider.
```

Answer:

1. Who or what creates the token?
2. Where is it stored?
3. Which identity can read it?
4. How does the app receive it at runtime?
5. Where could it leak?
6. How would you rotate it?
7. How would you revoke it during an incident?

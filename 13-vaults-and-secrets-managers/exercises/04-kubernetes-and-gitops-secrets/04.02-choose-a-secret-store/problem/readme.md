# Problem: Choose a Secret Store

Choose a store for each case:

1. A small Kubernetes dev cluster with one internal token.
2. A production app on AWS with RDS credentials.
3. A platform team issuing short-lived database credentials across clouds.
4. A GitOps repo that must contain encrypted desired state.
5. A CI pipeline that pushes images to a registry.

For each choice, name one risk to manage.

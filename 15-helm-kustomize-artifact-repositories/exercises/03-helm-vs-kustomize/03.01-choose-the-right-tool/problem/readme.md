# Problem: Choose the Right Tool

Choose Helm, Kustomize, or both for each case:

1. You are publishing a reusable internal app for many teams.
2. You need to change only namespace, image tag, and replica count per environment.
3. You are installing a third-party ingress controller.
4. You want Git to contain readable final environment overlays.
5. You need to publish a versioned deployment package to Nexus or Artifactory.
6. You need a one-off patch to add a label to all resources in prod.

Explain each choice in one sentence.

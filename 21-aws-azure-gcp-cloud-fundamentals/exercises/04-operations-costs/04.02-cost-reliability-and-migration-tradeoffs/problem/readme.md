# Problem: Cost, Reliability, and Migration Tradeoffs

Scenario:

```text
A team runs a public web app on AWS.
Leadership wants to know whether moving to Azure or Google Cloud would reduce cost.
The app uses containers, PostgreSQL, object storage, secrets, and logs.
```

Answer:

1. Which services would you map first?
2. What cost data would you collect before migration?
3. What reliability requirements must be preserved?
4. What hidden migration costs should you expect?
5. When is staying on the current provider the better choice?

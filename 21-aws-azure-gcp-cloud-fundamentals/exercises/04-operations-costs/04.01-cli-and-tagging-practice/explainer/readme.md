# Explainer: CLI and Tagging Practice

Cloud CLIs are useful for discovery and troubleshooting.

This lesson includes read-only command examples:

```text
cli-examples/aws-readonly.sh
cli-examples/azure-readonly.sh
cli-examples/gcp-readonly.sh
```

Tags and labels help with ownership, automation, policy, and cost allocation.

At minimum, track:

- environment
- owner
- application
- cost center

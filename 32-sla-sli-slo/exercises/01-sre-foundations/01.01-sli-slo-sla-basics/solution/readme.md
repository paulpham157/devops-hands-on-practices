# SLI, SLO, SLA Basics

| Statement | Classification |
| --- | --- |
| 99.9 percent successful checkout requests over 30 days | SLO |
| Successful checkout requests under 1000 ms divided by total checkout attempts | SLI |
| Monthly uptime commitment with service credit if missed | SLA |

An SLA is external and includes consequences, exclusions, and governance. It is not only a tighter target number.

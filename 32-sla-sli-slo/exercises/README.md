# Exercises: SLA, SLI, SLO

Work through these exercises after reading the lesson README.

## Exercise Map

1. `01-sre-foundations/01.01-sli-slo-sla-basics`
   - Separate measurement, objective, and contract concepts.

2. `01-sre-foundations/01.02-user-journeys-and-good-events`
   - Define good events from user journeys.

3. `02-objective-design/02.01-build-meaningful-slis`
   - Build SLIs that reflect user value instead of vanity signals.

4. `02-objective-design/02.02-slos-error-budgets-and-burn-rates`
   - Use objectives and budget consumption to guide operations.

5. `03-operationalization/03.01-alerting-and-error-budget-policy`
   - Connect SLOs to alerting and reliability policy.

6. `03-operationalization/03.02-sla-vs-internal-slo`
   - Keep customer contracts distinct from internal goals.

7. `04-production-reliability-management/04.01-reporting-dashboards-and-review-rhythm`
   - Operate service levels through review cadence and reporting.

8. `04-production-reliability-management/04.02-choose-the-right-sli-and-slo`
   - Choose practical indicators and targets for realistic services.

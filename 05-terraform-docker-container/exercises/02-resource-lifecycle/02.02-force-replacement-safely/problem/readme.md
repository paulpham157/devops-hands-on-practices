# Force Replacement Safely

## Task

Choose a change that replaces the container, such as changing `container_name`, or use the replacement workflow supported by your Terraform version.

Then answer:

1. What evidence in the plan shows replacement?
2. Why is forced replacement riskier in real infrastructure than in this local Docker lab?
3. What should be reviewed before replacing a production resource?

## Done When

You can recognize replacement in a plan and explain why replacement must be deliberate.

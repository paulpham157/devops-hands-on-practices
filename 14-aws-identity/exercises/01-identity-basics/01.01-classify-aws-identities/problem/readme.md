# Problem: Classify AWS Identities

Classify each case:

1. A developer logs into AWS console through company SSO.
2. An EC2 instance reads objects from S3.
3. GitHub Actions deploys to AWS without stored access keys.
4. A SaaS customer signs into your web app.
5. A break-glass owner identity controls the AWS account.
6. A legacy script uses an access key.

Use these choices:

```text
IAM Identity Center
IAM role
OIDC federated role
Cognito user
root user
IAM user
```

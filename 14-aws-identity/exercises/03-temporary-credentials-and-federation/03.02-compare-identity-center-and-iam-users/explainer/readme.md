# Explainer: Compare Identity Center and IAM Users

IAM Identity Center is for workforce access.

It centralizes access to multiple AWS accounts through permission sets and SSO.

IAM users are identities inside a single AWS account. They can have long-lived access keys and console passwords.

For humans, Identity Center is usually preferred because access is easier to grant, remove, audit, and align with the organization's identity provider.

IAM users should be exceptional and tightly controlled.

# Snyk vs Sonar

| Concern | Best Fit |
| --- | --- |
| Vulnerable npm dependency | Snyk |
| New code coverage gate | Sonar |
| Public security group rule in IaC | Snyk |
| Code smell and maintainability issue | Sonar |
| Container base image vulnerability | Snyk |

Both may report security issues in code, but their main operating roles are different.

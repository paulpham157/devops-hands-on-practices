# Create a GitLab CI Pipeline

## Task

Copy the sample GitLab CI file into a GitLab repository:

```bash
cp 08-github-actions-gitlab-ci/gitlab-ci/.gitlab-ci.yml .gitlab-ci.yml
```

Commit and push.

Inspect the pipeline and identify:

- The `test` stage.
- The `image` stage.
- The `publish` stage.
- The JUnit test report artifact.

## Done When

The `test` job passes and GitLab shows a test report artifact.

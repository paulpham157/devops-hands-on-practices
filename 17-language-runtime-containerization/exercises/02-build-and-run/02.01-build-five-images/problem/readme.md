# Problem: Build Five Images

Build each image:

```bash
./scripts/build-one.sh python-flask
./scripts/build-one.sh java-spring
./scripts/build-one.sh ruby-sinatra
./scripts/build-one.sh go-http
./scripts/build-one.sh rust-http
```

Then answer:

1. Which image built fastest on your machine?
2. Which image took longest?
3. Which build downloaded the most dependencies?
4. Which builds used multi-stage Dockerfiles?
5. What changed when you rebuilt an image without changing dependency files?

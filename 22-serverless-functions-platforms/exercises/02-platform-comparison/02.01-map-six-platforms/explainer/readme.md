# Explainer: Map Six Platforms

The diagram lists six serverless options.

The closest current names:

- AWS Lambda
- Cloudflare Workers
- Azure Functions
- Vercel Functions
- Netlify Functions
- Google Cloud Run functions, with Cloud Functions v2/API and 1st gen still visible in versions and tooling

They overlap, but they are not identical. Some focus on cloud event integrations. Some focus on edge and frontend workflows.

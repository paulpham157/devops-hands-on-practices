# Solution: Map Six Platforms

AWS Lambda is most AWS-native.

Cloudflare Workers is strongly edge-oriented.

Azure Functions is Azure-native.

Vercel and Netlify are frontend/site-adjacent platforms.

The Google Cloud naming note is that new function work should map to Cloud Run functions. Cloud Functions v2/API and 1st gen still appear in docs, APIs, and tooling, so older material may use those names.

# Explainer: Security, Observability, and Deployments

Production serverless needs the same discipline as servers.

Security:

- least privilege
- managed secrets
- input validation
- authentication
- dependency scanning

Observability:

- structured logs
- metrics
- traces
- error alerts
- cost alerts

Deployment:

- separate environments
- CI/CD
- rollback
- runtime pinning
- IAM review

# Senior Production Capstone

The capstone assesses curriculum-level production judgment. It does not certify employment seniority or replace sustained production ownership in a team.

Use it after completing the shared production-owner preparation core, the reference-system game day, and one specialization overlay. The brief, required artifacts, rubric, and defense process are public so assessment is reviewable and does not depend on hidden expectations.

## Capstone Package

- [System brief](system-brief.md)
- [Required artifacts](required-artifacts.md)
- [Review rubric](rubric.md)
- [Design defense protocol](design-defense.md)
- [Pilot calibration record](pilot-calibration.md)

The capstone is ready for structured pilot use once its prerequisites are met. Do not mark a learner as having passed until two reviewers have applied the rubric and the calibration record is updated.

# Curriculum Assessments

These assessments determine whether a learner has evidence for a level; they do not infer employment seniority. Reviewers should link every decision to an artifact, command output, or written decision.

- [Diagnostic and level exit rubrics](level-exit-rubrics.md)

Use the diagnostic before skipping a level. Use the exit rubric after the learner completes the level's shared core and required evidence.

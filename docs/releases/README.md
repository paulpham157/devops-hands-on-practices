# Learning Product Release Notes

Release notes are learner-facing records for significant curriculum milestones. Copy [TEMPLATE.md](TEMPLATE.md) to `vX.Y.Z.md`, complete it on `main`, and use the manual Draft Learning Product Release workflow only after [release readiness](../curriculum/release-readiness.md) is satisfied.

Routine lesson fixes do not require a release. A draft release must be reviewed before publication.

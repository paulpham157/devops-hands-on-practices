# Platform Product Metrics

| Metric | Definition | Baseline | Target | Collection source | Review owner |
| --- | --- | --- | --- | --- | --- |
| Adoption | percentage of eligible services using the path | | | | |
| Time to first valid deployment | time from repo creation to verified deployment | | | | |
| Change failure rate | deployments causing rollback, incident, or hotfix | | | | |
| Recovery time | median time from detection to verified mitigation | | | | |
| Support load | platform requests per service/team | | | | |
| Escape-hatch frequency | services leaving the path and reason | | | | |

Do not optimize adoption at the expense of safety or service outcomes. Record metric limitations and sampling assumptions.
